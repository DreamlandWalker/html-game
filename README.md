# 学习乐园 · 英语 RPG

一款融合了人教版新起点英语知识点与经典 RPG 战斗玩法的网页游戏。适合小学 1–6 年级学生边玩边学。

## 🎮 玩法概述

玩家选择职业（战士/法师/牧师/弓箭手），在 40 个随机生成的关卡中探索、答题、战斗。每答对一题才有机会出手，答错则被怪物攻击。

### 核心循环

```
探索关卡 ──→ 遭遇怪物 ──→ 答题/战斗中 ──→ 击败/被击败 ──→ 升级/探索下一关
```

## 🧩 游戏特色

### 战斗系统
- **回合制战斗**：玩家与怪物左右分屏布局，答题成功后选择攻击/防御/技能/用药水
- **答题机制**：每个怪物带一道英语题目（选择或填空），答对才能行动
- **动画反馈**：攻击/防御/治疗/受伤均有 CSS 关键帧动画
- **键盘快捷键**：1–4 选技能，Q 攻击，E 防御，R 用药水，1–4 选答案，Enter 确认
- **语音朗读**：答题正确后自动朗读英文句子（Web Speech API），支持重播

### 装备系统
- 4 个职业 × 8 个装备槽位 = 32 件套装装备
- 套装效果：2件/4件/6件/8件时激活不同增益
- 在营地铸造套装，在角色页装备

### 关卡系统
- 40 关随机生成地图（BFS 保证出口可达）
- 怪物掉落 6 种基础材料用于铸造
- 宝箱随机奖励

### 成就与成长
- 角色升级提升属性
- 学习/装备技能（主动+被动）
- 套装收集

## 🗺️ 页面导航

| 页面 | 说明 |
|------|------|
| `index.html` | 角色选择（初始页面） |
| `main_map.html` | 主地图，选择关卡、进入营地 |
| `game.html` | 关卡内探索 + 战斗 |

## 🛠️ 本地运行

无需安装任何依赖，直接打开浏览器：

```bash
# 任意方式启动一个本地服务器
python3 -m http.server 8080
# 或
npx serve .

# 浏览器访问 http://localhost:8080
```

> **注意**：部分功能（localStorage 存档、Web Speech API）需要 `http://` 协议，`file://` 下可能受限。

## 🔊 音频

- **背景音乐**：各页面自动播放，音量 12%
- **战斗音效**：攻击/技能/防御/受伤均有短音效，音量 15%–35%
- **语音朗读**：答题正确后朗读英文原句，支持重播

## 📜 技术栈

- 纯 HTML + CSS + JavaScript（无第三方库依赖）
- Web Speech API（语音朗读）
- CSS Animations（战斗动画）
- localStorage（存档）
